# my-utils

A small collection of utility functions.

## Installation

```bash
npm install my-utils
```

## Functions

- `add(a, b)` — adds two numbers
- `subtract(a, b)` — subtracts b from a

## Usage

```js
const { add, subtract } = require('my-utils')

add(1, 2)      // 3
subtract(5, 2) // 3
```
